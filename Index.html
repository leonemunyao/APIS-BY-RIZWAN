APIS

Application Programming Interface.....They are set of rules that allows different applications to communicate with one another .

Why API's

Accessibility of data

Types of APIS

  • Web APIS...accessed over web to allow communication btn servers
  • Library API Wrappers....provide a high level abstraction for complex functionalities...logic written by someone else
  • OS Apis.....Allow applications to intertact with underlying OS and hardware

We have an application that stores data and another one that needs the data stored
